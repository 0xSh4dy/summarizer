## Set the following environment variables

1. BASE_URL : The base url for the openai connector
2. MODEL : Model name (gpt-4, gpt-3.5-turbo)
3. OPENAI_API_KEY : Your API Key

## Build the binary
```bash
git clone https://github.com/0xsh4dy/pii_agent
cd pii_agent
cargo build
```

## Run
```bash
git clone https://github.com/0xsh4dy/pii_agent
cd pii_agent
cargo run
```
